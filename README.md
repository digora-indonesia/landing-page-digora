# Digora Academy Landing Page

Landing page modern dan responsif untuk Digora Academy, platform LMS untuk pendidikan dan pelatihan. Halaman ini dibuat untuk memperkenalkan brand, program belajar, fitur utama, profil founder, dan ajakan untuk mulai belajar secara gratis.

## Fitur Utama

- Desain modern dengan nuansa hijau, kuning, dan putih
- Tampilan responsif untuk desktop maupun perangkat mobile
- Hero section yang menarik dengan call-to-action yang jelas
- Menampilkan tiga jalur program: Pendidikan, Pelatihan Skill, dan Korporasi
- Menyertakan fitur LMS, proses belajar, serta profil founder
- Animasi reveal sederhana saat halaman discroll

## Struktur File

- [digora-academy-landing.html](digora-academy-landing.html) — file utama berisi seluruh markup, styling, dan interaksi halaman
- [README.md](README.md) — dokumentasi proyek

## Cara Menjalankan

Karena ini adalah halaman statis, Anda bisa langsung membuka file berikut di browser:

- [digora-academy-landing.html](digora-academy-landing.html)

Atau jalankan server lokal untuk preview yang lebih nyaman:

```bash
python3 -m http.server 8000
```

Lalu buka:

```text
http://localhost:8000/digora-academy-landing.html
```

## Kustomisasi

Anda bisa dengan mudah mengubah:

- Nama brand dan domain pada bagian header serta footer
- Teks hero, program, fitur, dan CTA
- Link LinkedIn founder
- Warna utama melalui variabel CSS di bagian root

## Teknologi yang Digunakan

- HTML5
- CSS3
- JavaScript vanilla

Landing page ini diharapkan membantu Anda mempresentasikan Digora Academy secara lebih menarik dan profesional.
